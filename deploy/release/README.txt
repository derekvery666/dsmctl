dsmctl CLI and local MCP release bundle

This archive contains:

- dsmctl: the command-line client for administrators, scripts, and CI.
- dsmctl-mcp: the local stdio MCP server for an AI client on this machine.
- LICENSE: Apache License 2.0.

dsmctl-mcp is not the Synology DSM package. The browser Admin console and
remote HTTP MCP service are delivered as dsmctl Gateway, either as a container
image or as the separate x86_64 Synology .spk release asset.

Documentation and release downloads:
https://github.com/derekvery666/dsmctl
https://github.com/derekvery666/dsmctl/releases

Verify the downloaded archive against SHA256SUMS before extracting it.
